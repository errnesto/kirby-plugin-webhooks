## Installation

### Download

Download and copy this repository to `/site/plugins/webhooks`.

### Git submodule

```
git submodule add https://github.com/errnesto/kirby-plugin-webhooks.git site/plugins/webhooks
```

### Composer

```
composer require errnesto/kirby-plugin-webhooks
```
